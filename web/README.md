# React-Redux-Starter

## Motivation
This project create starter for react with redux

## Packages (main)
- React
- React-Dom
- Redux
- Webpack2

## Using
```bash
git clone https://github.com/chutiphon-k/react-starter.git
npm install
npm run dev # For dev
npm run deploy:prod # For production
open url http://localhost:8080
```
